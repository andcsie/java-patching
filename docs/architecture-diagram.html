Agents & Tools

🔍
Analysis Agent
Impact analysis with AST parsing
Tools / Actions
analyze_impact Full AST-based impact analysis against release notes
explain_impacts Add LLM-generated explanations to impacts
get_release_notes Fetch JDK release notes between versions
get_security_advisories Get CVEs fixed between versions
🔧
Fixer Agent
AI-powered code fix generation
Tools / Actions
generate_fixes Generate LLM fixes for all impacts (with batching)
generate_single_fix Generate fix for a single impact
validate_fix Validate generated fix syntax
📝
Patcher Agent
Patch creation, testing & PR management
Tools / Actions
create_patches Generate unified diffs from fixes
apply_all_patches Apply patches to repository files
run_tests Run Maven/Gradle tests
create_pr Create branch, commit, push & PR
🎭
Orchestrator Agent
Full automated upgrade pipeline
Tools / Actions
full_upgrade Complete automated upgrade (all 5 steps)
🔄
Renovate Agent
Patch-level JDK version management
Tools / Actions
detect_version Detect JDK from pom.xml, build.gradle, etc.
get_available_patches Query Adoptium API for updates
preview_version_bump Preview version changes as diffs
apply_version_bump Apply version bump to build files
🍳
OpenRewrite Agent
Major version migrations with recipes
Tools / Actions
list_recipes List available migration recipes
analyze_migration Dry-run migration analysis
run_recipe Execute transformation recipe
suggest_migration_path Recommend upgrade path (8→11→17)
